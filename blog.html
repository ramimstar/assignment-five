Question-2: What is the difference between innerHTML, innerText, and textContent ?

      innerHTML
    1. Sets HTML content, including tags
    2. Parses and renders HTML inside the element
      innerText
    1. Sets only visible text inside an element
    2. Treats HTML tags as plain text and does not parse them
      textContent
    1. Sets all text inside an element, including hidden text
    2. Faster than innerText because it ignores styling and layout calculations

Question-3: What is event delegation in the DOM?

Event Delegation is a technique in JavaScript where a parent element handles events for its child elements, even if they are dynamically added later. Instead of adding event listeners to multiple child elements, you attach a single event listener to a common parent.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a concept in the DOM (Document Object Model) where an event that is triggered on an element "bubbles up" through its parent elements in the DOM tree

Question-5: How do you create, add, and remove elements using JavaScript?

  1. Creating an Element
  2. To create a new element in the DOM, you can use the document.createElement() method. This allows you to create a new HTML element as a JavaScript object that you can later manipulate.
  1. Adding an Element
  2. Once you've created an element, you can add it to the DOM using methods like appendChild(), insertBefore(), or append() depending on where you want to place the new element.
  1. Removing an Element
  2. To remove an element from the DOM, you can use removeChild() (on the parent element) or remove() (on the element itself).
